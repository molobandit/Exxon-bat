Ce que l'employeur doit remplir

Parcours guidé en 3 étapes : créer le chantier, inscrire l'équipe terrain, puis comparer le devis au métré signé et fixer le prix final. Les données ci-dessous sont déjà préremplies pour la démo.

Chargement de la simulation…
1

Créer le chantier

Page Chantiers — formulaire en haut de page.

L'employeur renseigne le nom, le client, l'adresse, le métier, le devis lié et le budget. Un code d'accès terrain est généré automatiquement — c'est ce code que l'employé saisit sur son téléphone.

Voir la page Chantiers
2

Inscrire l'équipe terrain

Page Équipe — un employé = identifiant + PIN.

L'employeur crée chaque intervenant avec prénom, nom, rôle et PIN (4 à 6 chiffres). L'identifiant employé est attribué automatiquement. L'employé se connecte ensuite avec ce couple identifiant / PIN + le code chantier.

Voir la page Équipe Vue employé terrain →
3

Valider le métré reçu

Page Métré — comparaison devis vs terrain.

Une fois le bulletin signé par le client sur chantier, l'employeur sélectionne le chantier, compare le devis initial au coût terrain, ajuste le prix final HT si le client a modifié le périmètre, puis clique sur Valider le métré ou Demander révision.
